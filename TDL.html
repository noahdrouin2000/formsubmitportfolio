Liste de Tâches

To Do List

Ce projet permet aux utilisateurs de faciliter leur quotidien avec une liste de tâche qui est à la fois interactive, utile, et amusante. Il est possible de créer des tâches, de les supprimer, et de les marquer comme complétées ou non. Il est également possible de filtrer les tâches selon leur état. Le site est conçu pour être utilisé sur mobile, tablette, et ordinateur. Les changements d'état des tâches sont enregistrés dans le "storage" local du navigateur, ce qui permet de garder les tâches même après avoir fermé le navigateur.

This project allows users to facilitate their daily life with a to do list that is both interactive, useful, and fun. It is possible to create tasks, delete them, and mark them as completed or not. It is also possible to filter tasks according to their state. The site is designed to be used on mobile, tablet, and computer. The state changes of tasks are recorded in the local browser storage, which allows to keep the tasks even after closing the browser.
